# Functional Programming

## File Structure
```
.
│-- index.js
└─── Data
    |-- Cars.js
    |-- Order.js
    |-- SpeciesTree.js
    |-- data.txt
```

## To run
- Install node.
- Create an empty folder.
- Run the following to create package.json.
```
npm init
```
- Add { "type": "module" } in the package.json file.
- Run the following to execute the script.
```
node index.js
```
